🛠️ Stack Tecnológica Recomendada

Linguagem & Runtime
Java 21 + Virtual Threads
Código imperativo simples com alta escalabilidade para I/O. Melhor equilíbrio entre desempenho e manutenibilidade.
Framework
Spring Boot 3.x
Ecossistema maduro, configuração simplificada, integração nativa com Azure através do Spring Cloud.
Cache
Caffeine
Cache em memória de alto desempenho, quase ótimo, com políticas avançadas de eviction e TTL configurável.
Orquestração
AKS + KEDA
Auto-scaling baseado em eventos, escala proativa baseada no consumer lag, não em métricas de recursos.
Mensageria
Event Hubs Premium
Latência previsível, isolamento de recursos, custo efetivo para alto volume com ingestão incluída.
Observabilidade
Application Insights
Rastreamento distribuído end-to-end, métricas customizadas, correlação automática logs-traces.
